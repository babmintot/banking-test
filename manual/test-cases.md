# Тест-кейсы ручного тестирования

---

## Общие предусловия

- Открыт браузер
- Запущено приложение по адресу:

  http://localhost:8000/?balance=30000&reserved=20001

- Доступная сумма:
  30000 − 20001 = 9999

---

## TC-01: Успешный перевод

**Описание:** Проверка успешного перевода при корректных данных

**Шаги:**
1. Нажать кнопку "Рубли"
2. Ввести номер карты: 1111222233334444
3. Ввести сумму: 1000
4. Нажать кнопку "Перевести"

**Ожидаемый результат:**
Перевод выполняется успешно, операция доступна

**Фактический результат:**
Перевод выполняется успешно

**Статус:** PASS

## TC-03: Граничное значение суммы

**Описание:** Проверка перевода на граничном значении доступной суммы

**Шаги:**
1. Нажать кнопку "Рубли"

3. Ввести сумму: 9099

**Ожидаемый результат:**
Перевод должен быть доступен, так как:
сумма + комиссия ≈ 9999 ≤ доступной суммы

"Недостаточно средств на счёте"
